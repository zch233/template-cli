# gupo-template

## 食用方法

### 快速体验

##### 安装

```
npm install gupo-template -g

// or

yarn global add gupo-template
```
##### 使用

```
// 进入项目终端
gupo-template [template-copy]
```

### 参数

```
$ gupo-template --help

  Usage
    $ gupo-template [本地文件目录名称]
    $ gupo-template [codeup 仓库地址, ssh 协议]
```
